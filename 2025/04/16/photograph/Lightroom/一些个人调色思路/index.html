调色不重要？调色很重要

发表于2025-04-16|更新于2026-02-24|摄影
|浏览量:

总结贴

示例图片有点糊…为了加载速度只能压缩画质了

日系蓝色胶片

基本

色温 - 色调 +

曝光适度 对比 - 白色色阶 - 黑色色阶 +

曲线

稍微下拉，亮部降暗部升

红色通道暗部下拉

混色器

绿色相 - 蓝色相 +

暖色饱和度 + 冷色饱和度（尤其蓝色） -

暖色明度 + 冷色明度（尤其蓝色） -

颜色分级

阴影偏青，高光偏黄绿

亮调春花

基本

色温 + 色调 -

曝光适度 对比 - 高光 - 阴影 + 白色色阶 - 黑色色阶 +

纹理 - 清晰度 + 去朦胧 -

鲜艳度 + 饱和度 -

曲线

稍微下拉，暗部升

混色器

橙绿色相 - 红蓝色相 +

暖色及蓝色饱和度 + 绿色饱和度 -

冷色明度 -

颜色分级

中间色调及阴影偏青，高光偏黄绿

暗调春花

基本

色温 - 色调 -

曝光适度 对比 - 高光 - 阴影 + 白色色阶 - 黑色色阶 +

纹理 + 清晰度 - 去朦胧 +

鲜艳度 + 饱和度 -

曲线

稍微下拉，暗部升

混色器

蓝橙色相 - 红绿色相 +

暖色饱和度 + 绿色饱和度 -

冷色明度 -

颜色分级

阴影偏青，中间色调偏黄绿，高光偏橙黄

蒙版

径向蒙版模拟打光（曝光+色温+），反向蒙版加深对比
